# Dossier Musiques

Ce dossier contient les fichiers audio de démonstration.

Pour ajouter de vrais fichiers audio :
1. Placez vos fichiers MP3/WAV ici
2. Mettez à jour la liste `musicLibrary` dans Create.jsx
3. Assurez-vous que les noms correspondent

## Fichiers requis actuellement :
- chill.mp3
- upbeat.mp3  
- ambient.mp3
- hiphop.mp3
- acoustic.mp3

## Alternative
Vous pouvez utiliser des URLs externes ou désactiver la bibliothèque prédéfinie.
